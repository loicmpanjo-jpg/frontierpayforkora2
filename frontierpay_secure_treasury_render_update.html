Treasury Accounts

Security Notice

Treasury account coordinates must never be exposed in frontend code, public repositories or client-side HTML. Store all banking coordinates inside Render environment variables or encrypted backend storage only.

  USD: {
    bank: process.env.PAYONEER_USD_BANK,
    account: process.env.PAYONEER_USD_ACCOUNT,

  EUR: {

    bic: process.env.PAYONEER_EUR_BIC
  }

};

Treasury Routing Logic

IF payout_country == Nigeria
    USE Kora

ELSE IF payout_type == Card
    USE Visa Direct

ELSE IF currency == USD
    USE USD Treasury

ELSE IF currency == GBP
    USE GBP Treasury

ELSE IF currency == EUR
    USE EUR Treasury

ELSE
    USE MANUAL FALLBACK
